¿Quienes Somos?

Somos un Grupo que se dedica al desarrollo de software en tecnologia de microsoft y software libre.

Nos dedicamos al desarrollo de aplicaciones de escritorio en (.net), desarrollo y diseño de Páginas Web interactivas, utilizando siempre las mejores y más nuevas tecnología.

Desarrollar grandes aplicaciones en un lenguaje estructurado y todavía experimentar ciclos de desarrollo Editar / Actualizar rápidas? De hecho, la profecía se ha hecho realidad. Obtener una locura productiva con herramientas de Dart, bibliotecas HTML lisas, polyfills cross-browser, y componentes web marco basado. Vamos a aprender cómo se puede desarrollar fácil y rápidamente aplicaciones web que funcionan entre navegadores en plataformas de escritorio y móviles. Le mostraremos cómo construir aplicaciones web modernas con componentes Web y vistas basadas en datos dinámicos sin tener que esperar a que los ciclos de compilación engorrosos. Haga un cambio, recarga golpear, y el auge, está listo para todos los navegadores modernos.
